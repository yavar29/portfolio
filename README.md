# Yavar Khan - Portfolio

Personal portfolio website showcasing AI/ML projects and software engineering work.

**Live:** [yavarkhan.me](https://yavarkhan.me)

## Tech Stack

- **Framework:** Next.js 16 (App Router, TypeScript)
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Fonts:** Inter + Plus Jakarta Sans
- **Deployment:** Vercel
